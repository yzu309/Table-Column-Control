## 介绍
列宽控制（Table Column Control）插件，是一个用于控制表格列宽的插件。
（插件仅在实时预览模式下生效，安装后，所有表格的列都会有一个列宽，表格将丧失自适应宽度能力）

插件会区分主窗口与新窗口：
	主窗口中，插件会给表格添加拖动手柄，允许拖动调整列宽。
	新窗口中，插件不会添加拖动手柄，仅会维护列宽。

插件会尽可能的尝试保留表格列宽，包括：
	新建列/删除列/移动列或行时，保留表格中其他列列宽；
	删除/新建表格时，保留其他表格列宽；
	复制表格时，优先匹配使用已有相同标题表格列宽；
	支持多窗口并发情况下的列宽更改同步；

可以在元数据cssclasses中输入不同数字，已启用不同模式：
	（每个数字需要单独写出）
	cssclasses=0，禁止插件生效，表格保持默认的自适应列宽；
	cssclasses=1，自适应铺满，表格总是会铺满编辑区；
	cssclasses=1 + 0，同时输入1和0，启用禁止拖拽且自适应铺满
	cssclasses=2，禁止拖拽，仅维护列宽
